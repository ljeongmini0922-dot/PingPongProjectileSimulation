# Ping-Pong Ball Projectile Motion Simulation

This project investigates the effects of air resistance and the Magnus effect on the projectile motion of a ping-pong ball.

## Models

- Vacuum projectile model
- Air resistance model
- Magnus effect model

## Files

- `simulation.py`: Python simulation code
- `report.tex`: Final LaTeX report
- `experiment pictures.png`: Experiment graph & Our experiment with pingpong ball 


## Physical Parameters

- Ball diameter: 40 mm
- Ball mass: 2.7 g
- Air density: 1.2 kg/m³
- Gravity: 9.81 m/s²
